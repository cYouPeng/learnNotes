new Object()  Object()是构造器  但Object不是类
用来构造对象，起到构造器作用的，首字母大写
Object Array String Pate Math Json RegExp

function person(){}
var p = new Person(); js中没有类的概念，只是特定代码的书写形式/技巧
同一个类的对象行为要一样，应该有相同的方法

每个构造器都有一个原型prototype
new ->构造器
调用->函数

写在构造器里的方法(1)和写在原型里的方法(2):
同：创建两个对象，各自方法的实现都相同
异：1改一个方法，不会影响另一个
    2改一个方法，其余皆改变
原型里的属性和方法被多个对象共享  共有的东西最好不要通过单个对象去改变

this+属性   原型+方法

一个对象同时有自身的属性或方法a和原型中的属性和方法a，优先找自身的a，没有再去原型中找

可以对内置对象改变和扩展方法
document对象类型 HTMLDocument

hasOwnprototype()只看自身是否有此属性，不往原型里找
in() 会往原型里找

js中静态方法，只能用类名调用，不能用对象名调用
静态方法直接写在构造器中

链式调用要求，函数返回当前对象

class里定义的函数是不可枚举的，直接表现是for in循环不到

__proto__ 原型指针 依次指向父类原型构造器

ES6的解构：let {nickName,sign,age} = res.result.user
此时单独写nickName相当于 res.result.user.nickName;
